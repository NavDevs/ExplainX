Installation Instructions

  1. Download the ZIP file from GitHub
  2. Extract the ZIP file to a folder
  3. Open Chrome and go to chrome://extensions
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked"
  6. Select the extracted folder
  7. Done! Start using ExplainX 🎉

Frequently Asked Questions

Is ExplainX really free?

Yes! ExplainX is 100% free. You only need your own OpenAI API key (pay-per-use, very cheap).

Do I need technical knowledge to install?

No! If you can follow 5 simple steps, you can install ExplainX. Takes less than 1 minute.

Is my data safe?

Absolutely. All processing happens locally. We only send selected text to your OpenAI API - nothing else.

Which websites work with ExplainX?

Any website! Wikipedia, news sites, blogs, documentation - if you can select text, you can explain it.

How do I get updates?

Check our GitHub Releases page for updates. Download the new version and reload the extension.
