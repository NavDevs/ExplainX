
# ExplainX - AI Powered Browsing Companion

**ExplainX** is a modern, high-performance Chrome Extension that provides instant AI-powered explanations for any text or code on the web. Built with a focus on speed, precision, and a premium "Glassmorphism" design.

![Logo](src/icons/icon128.png)

## ✨ Key Features

- **⚡ Instant Toggle**: A minimalist floating button that lives where you work.
- **🧠 Context-Aware Explanations**: Uses the Google Gemini Pro engine to explain complex concepts, code blocks, or technical jargon.
- **💬 Persistent AI Chat**: A dedicated sidebar to dive deeper into explanations with full conversational memory.
- **📂 Export & History**: Save your chat history and export insights to Markdown directly from the UI.
- **🌗 Smart Dark Mode**: Fully responsive theme that respects your system settings and host website style.
- **💠 Beautiful Design**: Premium Glassmorphism UI with smooth transitions and syntax-highlighted code.

## 🚀 Installation & Setup

### 1. Developer Mode
1. Clone this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (top-right toggle).
4. Click **Load Unpacked** and select the `dist` folder.

### 2. Configuration
- Click the ExplainX icon in your toolbar.
- Go to **Settings**.
- (Optional) Customize your default explanation persona (Student, Beginner Code, Professional).

## 🛠️ Development

ExplainX is built with a modern stack:
- **TypeScript**: For robust, type-safe extension logic.
- **Webpack**: To bundle assets and minimize production overhead.
- **Gemini Pro API**: High-end Large Language Model for intelligent responses.
- **Marked & Highlight.js**: For rendering professional-grade formatted explanations.

### Commands
- `npm run build`: Compile the production-ready `dist` folder.
- `npm run dev`: Build and watch for changes during development.

## 🔏 Privacy
ExplainX values your data privacy. We only send the text you explicitly highlight or type into the chat to the AI provider (Google Gemini) for analysis. We do not store your data on external servers.
